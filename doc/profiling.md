# Profiling 

### Rails Console Approach 

Inspecting GTFSGraph for memory usage during level 1 imports.
- GC::Profiler.enable 
- GC::Profiler.clear 
- graph = GTFSGraph.new(feed, feed_version) 
- graph.cleanup 
- graph.create_change_osr 
- GC::Profiler.report
